<objective>
Ship the YouTube source class — the multimodal-secondary pipeline. Pulls auto-captions + downloads the 720p video + ffmpeg samples one frame per 30 seconds + SKILL.md instructs Claude to Read each frame and produce a vision description. Merges captions + frame descriptions into `multimodal_segment` chunks. NO Whisper.

Purpose: RESEARCH-07 demands multimodal ingestion specifically — captions alone lose information visible on screen. The decisive spike-002 moment (effect list visible on screen at t=7:00 while captions said only "28 different effects types") is the load-bearing justification.
Output: A `youtube.py` source class + the `parse_vtt.py` + `yt_pipeline.py` helpers + reference doc + VTT fixture + tests.
</objective>

<interfaces>
Chunk types this source class emits (from `skills/patchbay-ingest/references/chunk-schema.md`):
- `multimodal_segment`: content is `{timestamp, frame, frame_description, caption_text, what_audio_misses}` — the spike-002c primary chunk type
- `transcript`: content is `{start_time, end_time, text}` — caption window
- `external_resource`: content per the standard external_resource shape — populated when video description contains gear-relevant external URLs

YouTube pipeline (locked, from spike findings):
1. `yt-dlp --write-auto-subs --skip-download --sub-format vtt -o <tempdir>/video <url>` → produces `<tempdir>/video.<lang>.vtt`
2. `yt-dlp -f "bestvideo[height<=720][ext=mp4]+bestaudio[ext=m4a]/best[height<=720]" --merge-output-format mp4 --output <tempdir>/video.mp4 <url>`
3. `parse_vtt(<tempdir>/video.<lang>.vtt)` → list of `{start, end, text, timestamp_display}` windows
4. `ffmpeg -i <tempdir>/video.mp4 -vf "fps=1/30" -q:v 3 <tempdir>/frames/frame_%03d.jpg`
5. SKILL.md tells Claude to Read each `frame_*.jpg`, write a `frame_description` string, and pair it with the caption window at the same timestamp index.
6. `build_multimodal_chunks(caption_windows, frame_descriptions, gear_ctx, video_url)` zips them into chunks with deep_link `<video_url>&t=<seconds>s`.
</interfaces>
